**OpenParty** is a web engine for **multiplayer chat games**, designed for speed and comfort.
This framework is proudly powered by **Node.js** and uses **Angular.js**, **Express** and **Socket.io** modules.

OpenParty is an open-source project hosted on GitHub_.
We'll welcome all suggestions or pull requests (translations, new features, fixes...) !

.. _GitHub: https://github.com/Lesterpig/openparty
